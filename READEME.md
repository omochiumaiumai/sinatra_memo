# メモアプリ
sinatraを使ったメモアプリケーションです。
## できること
+ メモを追加
+ メモを見る
+ メモを削除
+ メモを編集
## 立ち上げ手順

1. リポジトリをクローンする

`git clone https://github.com/omochiumaiumai/sinatra_memo.git`

2. sinatra_memoディレクトリに移動

`cd sinatra_memo/`

3. Gemfileからgemをインストール

`bundle install`

4. コマンドを実行

`bundle exec ruby app.rb`
